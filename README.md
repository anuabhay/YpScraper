# YpScraper
## How to RUN

* To run you need following
    * GeckoDriver v0.20.1
    * Firefox 53 and greater
    
How to run the Jar

    java -cp -Dwebdriver.gecko.driver=$path_to+gekodriber/geckodriver com.hes.YPScraper -t types.json -l locations.json
